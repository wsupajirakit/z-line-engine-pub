# Z Line Engine (Public Structure Repo)

This is a fully self-initiated and self-developed project, not derived from client work.

> หมายเหตุ: repository นี้เป็นเวอร์ชันที่ทำการ redacted แล้ว
> ไม่สามารถเปิดเผย source code ทั้งหมดของระบบจริงได้
> จึงแสดงเฉพาะโครงสร้างโปรเจกต์ + โค้ดตัวอย่างเชิงสถาปัตยกรรม (skeleton) สำหรับอ้างอิงเท่านั้น

## What is included

- Folder/File structure ของระบบ
- ขอบเขตโมดูลหลัก (controllers, services, models, middleware, routes)
- ตัวอย่าง flow ระดับสูงแบบไม่เปิดเผย business logic ภายใน

## What is excluded

- Core business logic
- Internal implementation details
- Sensitive integrations / production secrets
